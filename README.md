# ✈️ Customer Satisfaction Dashboard (Excel)

## 📊 Project Overview
This project presents an interactive Excel dashboard analyzing airline customer satisfaction using the Invistico Airline dataset. The dashboard provides insights into passenger demographics, travel behavior, and service quality metrics.

## 🎯 Objective
To analyze customer satisfaction patterns and identify key factors influencing passenger experience using data visualization techniques in Excel.

---

## 🧩 Dataset
- Source: Invistico Airline Dataset
- Records: ~120K+ passengers
- Features include:
  - Gender, Age
  - Travel Class (Business, Eco, Eco Plus)
  - Flight Distance
  - Satisfaction (Satisfied / Neutral / Dissatisfied)
  - Service ratings (WiFi, Food, Comfort, etc.)

---

## 📈 Dashboard Features
- Total Passengers, Gender Split, Travel Class Distribution
- Satisfaction analysis across:
  - Gender
  - Age groups
  - Flight distance
- Service rating impact on satisfaction:
  - Seat comfort
  - Food & Drinks
  - WiFi service
  - Baggage handling
- Interactive slicers for dynamic filtering

---

## 🛠️ Tools & Techniques
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Data Cleaning & Transformation
- Interactive Dashboard Design

## 🔍 Key Insights
- Business class passengers show higher satisfaction rates
- Long-distance flights have varying satisfaction trends
- Service quality (WiFi, Seat Comfort) strongly impacts satisfaction
- Age and travel purpose influence customer experience
